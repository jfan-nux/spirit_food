# Spirit Food ID Card Template

## Overview

The ID card is the final output of the Spirit Food quiz. It's generated by AI (Portkey/OpenAI) based on the determined dish type, cuisine, and alignment. For V1, this is text-only.

## AI Generation Prompt

```python
ID_CARD_PROMPT = """
Generate a Spirit Food ID card for the following combination:

SPIRIT FOOD: {alignment_adjective} {cuisine} {dish_type}
- Dish Type: {dish_type}
- Cuisine: {cuisine}
- Time Alignment: {time_axis} ({time_percent}%)
- Adventure Alignment: {adventure_axis} ({adventure_percent}%)

Based on the food_personalities.md traits for {dish_type}, generate:

1. **Creative Title** (2-4 words)
   - Should capture the essence of this specific combination
   - Examples: "The Midnight Broth Whisperer", "Dawn's Crispy Champion", "The Unhinged Flavor Alchemist"
   - Incorporate the time alignment (morning = dawn/sunrise/early, midnight = nocturnal/night/midnight)
   - Incorporate the adventure level (comfort = cozy/warm/steady, adventurer = wild/bold/chaotic)

2. **Three Strengths** (1 sentence each)
   - Exaggerated positive traits specific to this food+cuisine combo
   - At least one should reference the cuisine's cultural aspect
   - At least one should reference the time alignment
   - Make them funny and slightly absurd

3. **Three Weaknesses** (1 sentence each)
   - Humorous food-related flaws
   - Self-deprecating but endearing
   - Can reference common complaints about this food type

4. **Two Signature Quotes** (what this food would say)
   - First-person statements from the food's perspective
   - Should reflect the alignment personality
   - Slightly philosophical but also silly

5. **Hidden Talent** (1 phrase)
   - An unexpected ability this food has
   - Should be whimsical and specific

6. **Three Peer Reviews** (from other foods)
   - Format: "[Review text]" - [Food Name]
   - Choose reviewer foods that have interesting relationships with this food
   - Can be admiring, jealous, confused, or competitive
   - Should be funny and reveal something about both foods

TONE: Whimsical, absurdist humor, zodiac-reading energy, playfully exaggerated
STYLE: Like BuzzFeed meets horoscope meets food critic
LENGTH: Keep each item concise but impactful

Return in JSON format:
{{
  "title": "...",
  "strengths": ["...", "...", "..."],
  "weaknesses": ["...", "...", "..."],
  "quotes": ["...", "..."],
  "hidden_talent": "...",
  "peer_reviews": [
    {{"text": "...", "reviewer": "..."}},
    {{"text": "...", "reviewer": "..."}},
    {{"text": "...", "reviewer": "..."}}
  ]
}}
"""
```
